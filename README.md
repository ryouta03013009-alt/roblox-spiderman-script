# Roblox Spider-Man Mobile Script

モバイル（スマートフォン）専用のRoblox用スパイダーマンスクリプト。一人称視点でレティクルを表示し、タッチ操作で糸を出して建物に登ったりできます。

## 機能

- **一人称視点**: プレイヤーの視点からゲームをプレイ
- **モバイル最適化**: スマートフォンのタッチスクリーンに完全対応
- **レティクル表示**: 画面中央に緑色のレティクルを表示
- **糸発射**: 画面をタップして糸を発射し、対象に向かって移動
- **壁登り**: スワイプで壁を登ることができます

## インストール方法

1. Robloxゲームを開く  
2. `StarterPlayer` → `StarterCharacterScripts` に新しい **LocalScript** を作成  
3. `spiderman-mobile.lua` のコードをコピー＆ペースト  
4. ゲームをプレイしてテスト

## 操作方法

| 操作 | 機能 |
|------|------|
| **画面をタップ** | 糸を発射（タップした位置に移動） |
| **上方向へスワイプ** | 壁登り |
| **下方向へスワイプ** | 通常移動に戻す |

## カスタマイズ

スクリプト内の設定を変更できます：

```lua
local ROPE_SPEED = 100        -- 糸で引っ張られる速度
local ROPE_LENGTH = 100       -- 糸の最大長
local SWING_DAMPING = 0.95    -- スイング時のダンピング
local GRAVITY = 196.2          -- 重力の値
```

## 必要な環境

- Roblox Studio
- Lua（Roblox Lua）
- モバイルデバイス（iOS/Android）
